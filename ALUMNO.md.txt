## Los 3 conceptos clave de Git

1. **Repositorio:** [es una carpeta en la nube que guarda tu codigo y toda su historia]
2. **Commit:** [guardar el estado actual de los arvhivos, le pones una nota breve explicando algun cambio que se hizo y si algo falla en ek futuro puedes voler a ese commit]
3. **Rama:** [es una copia paralela de tu proyecto para hacer pruebas sin romper el codigo principal]

## Los comandos que aprendí
git add: Agrega los archivos al área de preparación (staging) para incluirlos en el próximo commit.
git commit: Guarda los cambios agregados con un mensaje descriptivo.
git push: Sube los commits al repositorio remoto (por ejemplo, en la nube).
git pill: Descarga y actualiza tu repositorio local con los cambios del remoto.
